# 萌宠日记

一个专注于宠物日常行为记录的温馨工具，帮助您轻松管理多只宠物的成长点滴。

## 🌟 核心理念

回归本质，记录陪伴。我们致力于为铲屎官提供一个纯净、无广告、无社交压力的宠物日记空间。无论是喂食、散步，还是生病、就医，这里的每一条记录都是您对爱宠深情的见证。

## ✨ 主要功能

- **🐾 多宠档案管理**：支持添加多只宠物（狗狗、猫咪、鸟类等），详细记录每只宠物的品种、性别、生日、体重等基本信息。
- **📝 全方位行为记录**：
  - **生活记录**：喂食、饮水、零食、睡觉、体重。
  - **日常活动**：遛弯、玩耍、训练、洗澡、心情。
  - **健康维护**：疫苗、驱虫、就医、便便/嘘嘘监测。
- **📅 智能待办提醒**：不再错过疫苗接种、体内外驱虫等重要日子。
- **📊 数据统计分析**：按月度生成行为统计报表，直观查看宠物的成长趋势和生活规律。
- **📸 影像日记**：每条记录均支持上传图片，定格每一个可爱瞬间。
- **☁️ 云端数据同步**：支持将数据同步至服务器，更换设备也不用担心丢失珍贵记录。

## 🚀 技术架构

- **前端**：`uni-app` (基于 Vue 3)，支持微信小程序及多端运行。
- **后端**：`Node.js` + `Express`。
- **数据库**：`SQLite` (轻量、高效，适合私有化部署)。
- **存储**：支持本地与服务器双向存储。

## 📂 项目结构

```text
weileme/
├── pages/                 # 前端页面
│   ├── index/index.vue    # 首页（今日摘要、快捷记录、我的宠物）
│   ├── pet/               # 宠物管理（添加宠物、宠物详情）
│   ├── record/            # 记录管理（添加记录、历史列表、统计数据）
│   └── reminder/          # 提醒中心
├── utils/                 # 工具类（API 请求、配置）
└── server/                # 后台服务代码
```

## 🛠️ 快速开始

### 1. 启动后端
```bash
cd server
npm install
npm start
```
服务默认运行在 `http://localhost:3000`。

### 2. 启动前端
1. 使用 **HBuilderX** 打开本项目。
2. 配置 `utils/config.js` 中的 `BASE_URL` 指向您的服务器地址。
3. 运行到微信开发者工具或真机。
